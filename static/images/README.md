# 画像フォルダーについて

このフォルダーにブログ記事で使用する画像を配置すること。

## 使用方法

1. 画像ファイルをこのフォルダーに配置
2. マークダウンファイルで以下の形式で参照：
   ```markdown
   ![画像の説明](/images/ファイル名.jpg)
   ```

## 推奨ファイル形式
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- SVG (.svg)

## ファイル命名規則
- 英数字とハイフンを使用
- 日本語ファイル名は避ける
- 例：`sample-image.jpg`、`blog-post-01.png`
